## Description
Interactive dashboard to analyze university admission, enrollment, retention and student satisfaction data from 2015 to 2024. Includes filters by year and term.

## Files
| File | Description |
|---|---|
| `app.py` | Streamlit dashboard |
| `requirements.txt` | Required libraries |
| `university_student_data.csv` | Dataset |
| `actividad1_dashboard.py` | Exploration and visualizations |

## How to run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```
